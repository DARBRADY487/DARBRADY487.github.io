How To Use

Map Page

On the map page, you see all the game-events around you, up to a 20-Mile radius, which are coming up in the next 7 days. You can pan the map and zoom in and out to discover more/less game events.

Each game icon on the map represents one specific game-event and tapping the icon gets you to that game-event page.

If there are too many games on the map, use one of the filters to narrow down the games, dates or distance.

List Page

Here are listed all the game-events around you, up to a 20-Mile radius, which are coming up in the next 7 days. All the games on the list are arranged chronologically.

Each game tab on the list represents one specific game-event and tapping the tab gets you to that Game-Event page.

If there are too many games on the map, use one of the filters to narrow down the games, dates or distance.

Game-Event Page

On this page you can:

  • See all the relevant information about this game-event.
  • See who is the host and who already applied to play.
  • Tap on the photos of the host and players to see their profile page.
  • Read more about the game and any other special comments the host added about it in the message board at the bottom of the page.
  • Apply to a game (or cancel your application if you need to).
  • See the exact address of the game-event, only If you were approved.

Notes for Hosts

After publishing your game-event, this page will enable you to:

  • Approve players who applied to join (or disapprove them later on).
  • Answer players questions in the message box at the bottom of the event page.
  • Edit the information about your game-event if you need to.

Tapping the [X] on the top-right will clear all the filter choices you made, and will close the filter to get you back to the map or list.

Tap the distance from your location to filter the results on the map and list.

Tapping the [X] on the top-right will clear all the filter choices you made, and will close the filter to get you back to the map or list.

Settings Page

To host your own game, tap the type or the specific game you would like to host and set all the relevant points about it; number of players you are looking for, your address, the date and time, and any other information that may help other players:

  • Looking for beginner or advanced players?
  • Only for singles/couples?
  • Equipment needed for the game?
  • People who only want to watch the game – welcome?
  • Event finish time?
  • Kids/pets/monsters allowed? Are on premises?
  • Special directions on how to get there? Bus/train?

Notes:
The ‘Done’ button that publishes your game-event will not be enabled unless you fill out all the fields on this page.

You can set an event date up to 2 weeks ahead, but as the app only shows games that are 1 week ahead, you will only be able to see it in the Map or List pages when it enters the coming week.

As there are too many games to add to the game-picker list, we enabled a general ‘Other’ choice for each type of games, and an info box where you can elaborate on it more.

To get back to the map or list tap the [X] at the top-right.
